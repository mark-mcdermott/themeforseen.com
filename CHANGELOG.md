# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-01

### Added

- Test suite with Vitest and Playwright
- Printful store ID support for multi-store accounts
- Cart and checkout functionality for merchandise store
- Visual previews on favorites page
- Real-time updates for favorites

### Changed

- Use theme-forseen from npm instead of local link
- Copy theme data locally for production Cloudflare builds

### Fixed

- Svelte 5 reactive state warnings
- Neon 500 cold start issues
- Fullscreen mode on site preview
- Variable name bugs for fonts
- Critical bugs in Favorites functionality
- UI issues in paid features
- Palettes page Input and Label components
